# Projetos do Estágio – PagBank

Este repositório tem como objetivo **registrar e organizar meu progresso de aprendizado durante o estágio no PagBank**.

Aqui serão armazenados os **projetos e exercícios** desenvolvidos ao longo do estágio, servindo tanto como histórico de evolução técnica quanto como material de consulta futura.

---

## 🎯 Objetivo do Repositório

- Acompanhar minha evolução técnica ao longo do estágio
- Centralizar projetos e atividades práticas desenvolvidas
- Praticar boas práticas de versionamento com Git e GitHub
- Servir como portfólio pessoal de aprendizado

---

## 📂 Organização

Os projetos serão organizados de forma progressiva, podendo variar conforme o conteúdo abordado, por exemplo:

- Projetos individuais
- Exercícios de estudo
- Testes de conceitos
- Pequenas aplicações ou APIs
- Scripts e experimentações

Cada projeto poderá conter sua própria documentação quando necessário.

---

## 🛠️ Tecnologias

As tecnologias utilizadas neste repositório podem incluir, mas não se limitam a:

- Linguagens e frameworks utilizados no estágio
- Ferramentas de desenvolvimento
- Boas práticas de versionamento e organização de código

*(As tecnologias específicas serão detalhadas conforme os projetos forem adicionados.)*

## 🚧 Observações
